I found a bug! How I can report it?

Yes! You can report bugs on the GitLab issue tracker. Also you can report directly to emulator developers at IRC #cc.ru@irc.esper.net

Where I can find the emulator hotkeys, usage instructions etc?

Emulator GUI hotkeys always shown at the bottom of the window

But there was a some hidden hotkeys:

F1 UI layout debbuger

F2 FPS Counter

F3 Reloads workspace

F4 Emulator perfomance profiler window

How to change screen scaling?

Just drag drag the bottom-right corner of the window to resize screen scale. Screen are resized by steps like: 1/1 (100%); 1/2 (50%); 1/4 (25%); ... If you want to SCALE above 100% - you need to press Shift key while dragging
